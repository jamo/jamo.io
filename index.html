Work Related Projects

TestMyCode

The core developer and maintainer of the TestMyCode projects. Test My Code ("TMC") is a suite of tools that has been designed to make the life of programming instructors and students easier, as well as provide beneficial input for computing education researchers. TMC integrates to real IDE allowing students to immediately start working in real development environments while learning programming. From IDE, it also gathers detailed information regarding students' programming process for researchers to use.

TMC is available in both self hosted and SAAS, there are at least three universities and a few university of applied sciences using the SAAS version that I host. On average day during peak hours we get about 20 QPS to the Rails app.
Even thou we are hosting on commodity hardware, we've been able to maintain four nines availability

Mentionable achievements

I've done lots of work and cool features for the TestMyCode project. To mention a few:

  • in addition to NewRelic, I've setup ELK stack in distributed manner and a neat dashboard of all of the important metrics to improve visibility and make future planning easier.
  • Improved performance by better utilizing caching (both client and server side), optimizing SQL queries, and usage of rails features.
  • Optimized deployment, and set up proper staging and canary instances.
  • Currently I'm leading one of the biggest refactorings of the client side application logic
